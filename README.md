# StateCraft

This small tool is a visual storyboard editor,  
hopefully providing you with all the tools needed to create wonderful storytelling sessions.

It supports a wide selection of visualisation formats:
	* Text
		* Plaintext, Markdown, [BBCode](https://docs.godotengine.org/en/stable/tutorials/ui/bbcode_in_richtextlabel.html#reference)
	* Images
		* supports ".jp(e)g", ".png", ".bmp", ".svg", ".webp"
		* no (current) support for ".heif", ".tif", ".gif", ".jxl", ".ico", ".psd", ".raw" or proprietary formats
	* Audio
		* supports ".mp3", ".ogg", ".wav"
	* Video
		* supports ".mp4" and many others
